# Место — интерактивная страница с карточками

Учебный проект. Реализована регистрация, авторизация, добавление/удаление карточек, лайки, редактирование профиля.

## Технологии (фронтенд)
- React
- JavaScript
- Адаптивная вёрстка

## Бэкенд (использовался готовый API)
- Express.js
- MongoDB

## Моя работа над проектом
- Вёрстка и адаптив главной страницы и попапов
- Логика работы с API (загрузка карточек, профиля)
- Реализация лайков, удаления, добавления карточек
- Настройка валидации форм




Стартовый набор для выполнения проектной работы Место
